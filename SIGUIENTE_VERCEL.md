## 🚀 Siguiente Paso: Conectar con Vercel

### Opción 1: Desde Vercel Dashboard (Recomendado)

1. **Ir a Vercel**: https://vercel.com/dashboard
2. **Iniciar sesión** con la cuenta de tu empresa
3. Click en **"Add New..."** → **"Project"**
4. Verás una lista de tus repositorios de GitHub
5. Busca **"kommo-widgets-hub"**
6. Click en **"Import"**
7. **Configuración del proyecto:**
   - **Project Name**: `kommo-widgets-hub` (o el que prefieras)
   - **Framework Preset**: `Other`
   - **Root Directory**: `./` (dejar por defecto)
   - **Build Command**: (dejar vacío)
   - **Output Directory**: (dejar vacío)
8. Click en **"Deploy"**
9. Espera 1-2 minutos
10. ¡Listo! Te dará una URL como: `https://kommo-widgets-hub.vercel.app`

### Opción 2: Desde Vercel CLI

```powershell
# Instalar Vercel CLI (si no lo tienes)
npm install -g vercel

# Ir a la carpeta
cd c:\Desarrollo\kommo

# Desplegar
vercel

# Para producción
vercel --prod
```
