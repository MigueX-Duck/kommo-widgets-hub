# 🚀 Pasos para Subir a GitHub y Desplegar en Vercel

## ✅ Completado
- [x] Inicializar repositorio Git
- [x] Crear primer commit

## 📋 Siguiente: Crear Repositorio en GitHub

### Paso 1: Crear Repositorio

1. Ve a: https://github.com (cuenta de tu empresa)
2. Click en **+** (arriba derecha) → **"New repository"**
3. **Completa:**
   - **Owner**: Tu empresa/organización
   - **Repository name**: `kommo-widgets`
   - **Description**: `Servidor centralizado de widgets personalizados para Kommo CRM`
   - **Visibilidad**: **Private** (recomendado)
   - **NO marques**: "Add a README file"
   - **NO marques**: "Add .gitignore"
4. Click en **"Create repository"**

### Paso 2: Copiar URL

GitHub te mostrará una página con la URL del repositorio. Copia la URL, será algo como:

```
https://github.com/TU-EMPRESA/kommo-widgets.git
```

### Paso 3: Pegar la URL Aquí

Una vez que tengas la URL, pégala en el chat y continuaré con:
- Conectar el repositorio remoto
- Hacer push del código
- Conectar con Vercel
- Desplegar

---

## 🎯 Después del Push

Una vez en GitHub, conectaremos Vercel:

1. Ir a https://vercel.com/dashboard
2. Click en "Add New..." → "Project"
3. Importar el repositorio `kommo-widgets`
4. Deploy
5. ¡Listo!
